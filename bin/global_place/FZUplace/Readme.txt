Please have all the respective benchmarks (ISPD05, ISPD06) located in the current directory

Command-line format:
./FZUplace -aux benchmark/benchmark.aux -util [density] -nolegal -nodetail

An example for running our binary:
./FZUplace -aux adaptec1/adaptec1.aux -util 1 -nolegal -nodetail

Noticed that FZUplace only account for global placement
